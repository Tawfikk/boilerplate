# boilerplate

Swift 3+ and Using MVVM architecture(Rxswfit + Moya) to implement Github client demo.

## Features

*  Swift 3+
*  Rxswift
*  Moya (networking layer)
*  MVVM architecture implement
*  TableView Pagination 
*  RxDataSources
*  ObjectMapper (Object mapping)

## Setup

```
pod install
```

## Requirements

* iOS 8.0+ / Mac OS X 10.9+
* Xcode 8.0+
